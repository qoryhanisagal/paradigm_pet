## ‚ö´Ô∏è Overview

Paradigm Pet Professionals is a responsive, modern website designed to connect pet owners and professionals with expert advice (from "Pexperts") and personalized consultation services. The site provides resources across various pet types, including dogs, cats, fish, and more, focusing on **evidence-based care** for the overall well-being of companion animals.

This project showcases a clean, professional design using a friendly color palette and leverages modern web technologies for a smooth user experience.

---

## ‚ú® Features

- **Responsive Design:** Fully responsive layout ensures optimal viewing across desktops, tablets, and mobile devices.
- **Hero Section:** Engaging homepage hero with a strong call-to-action for scheduling consultations.
- **Service Offerings:** Clearly defined sections for One-on-One Consultations, Group Presentations, and Virtual Care.
- **Pet Care Resources:** Dedicated sections for basic care advice (Dog, Cat, Fish, Bird, Small Animals).
- **Adoption Guidance:** Links and resources to reputable pet adoption organizations (ASPCA, Shelter Pet Project, etc.).
- **Quick Consultation Form:** A simple, embedded form for users to request a consultation.
- **Interactive Search Modal:** A responsive modal for site-wide search functionality.
- **Professional Styling:** Utilizes a custom CSS stylesheet with a focus on readability and accessibility.

## üõ†Ô∏è Technology Stack

The project is built entirely with standard web technologies and leverages external libraries for enhanced functionality and styling.

## ‚ö´Ô∏è Design and Customization

The design is built around a distinct, defined color palette and typography.

| Functional Role       | Hex Code           | Description                                                                                |
| :-------------------- | :----------------- | :----------------------------------------------------------------------------------------- |
| **Primary Brand**     | `#003B6D`          | Main branding color, used for headers, strong text, and navigation.                        |
| **Secondary Brand**   | `#6699CC`          | Used for backgrounds, shadow effects, and secondary brand elements.                        |
| **Main Accent/CTA**   | `#FDB515`          | Bright, high-visibility color for calls-to-action, highlights, and icons.                  |
| **Neutral Dark Text** | `#333` / `#676767` | Used for body text and secondary text.                                                     |
| **Light Neutrals**    | `#BDBDBD`          | Used for light borders, separators, and subtle backgrounds (e.g., "Coming Soon" elements). |

**Key Customization Points:**

- **Styling:** All custom styles are currently located within the `<style>` block in `index.html`. For larger development, it's highly recommended to move this into an external `style.css` file.
- **Color Consistency:** The project now uses the specific color `#BDBDBD` for light elements (like borders and the "Coming Soon" button) and `#676767` for some dark text/neutral elements for brand consistency.
- **Logo/Images:** Update the `img/logo-icon.png` and `img/hero.png` paths with your actual asset files.
